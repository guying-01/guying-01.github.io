关于js操作改变input值，不触发onchange事件

input onchange是失去焦点以后触发的，我们通过js去操作它一直没有获得焦点，也就不会触发。
同理，也不会触发oninput，因为oninput是用户修改内容时才会触发。解决方法就是手动调用。
